# WhisperVault Frontend

Frontend application for **WhisperVault**, built with React and Vite.

This project provides the user interface for the WhisperVault platform, enabling users to authenticate, create posts, manage profiles, and interact with the backend API.

**Note:** all the posts, comments, user data and any other things may be deleted without notice.

---

### This project is for educational and personal use only.
*warning: Please do not share personal data. you are not required to use your personal/real identity (name/email) to try the demo site*
- Live demo link (https://whisper-vault.netlify.app/)

---

## Tech Stack

- React
- Vite
- JavaScript (ES6+)
- React Router
- Context API
- CSS

---

## Features

- User authentication (login & signup)
- Create, edit, and manage posts
- Profile and settings management
- Protected routes
- Responsive UI
- API integration with backend service
